# KikoGame

Игра для контроля знаний об искусственном интеллекте сотрудниками компании.

## Описание

KiKoGame - это интерактивная игра, где игрок управляет ракетой, нажимает на здания и проходит тесты (20 вопросов) по искусственному интеллекту.

## Технологии

- Python 3.x
- Pygame

## Установка и запуск

1. Установите зависимости:
```bash
pip install pygame
```

2. Запустите игру:
```bash
python main.py
```

## Управление

- **Стрелки** - управление ракетой
- **Пробел** - пауза
- **Мышь** - клик по зданиям для прохождения тестов

## Деплой

## Структура проекта

- `main.py` - главный файл игры
- `spaceship.py` - класс ракеты
- `Test.py` - система тестирования
- `departments_data.py` - данные о департаментах и вопросах
- `PICS/` - изображения и звуки игры
