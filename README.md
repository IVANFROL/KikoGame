# KikoGame

Игра для контроля знаний об искусственном интеллекте сотрудниками компании.

## Описание

KiKoGame - это интерактивная игра, где игрок управляет ракетой, нажимает на здания и проходит тесты (20 вопросов) по искусственному интеллекту.

## Технологии

- Python 3.x
- Pygame

## Установка и запуск

1. Установите зависимости:
```bash
pip install pygame
```

2. Запустите игру:
```bash
python main.py
```

## Управление

- **Стрелки** - управление ракетой
- **Пробел** - пауза
- **Мышь** - клик по зданиям для прохождения тестов

## Деплой в браузере

### Replit (Рекомендуется)

Для быстрого деплоя в браузере используйте Replit:

1. Зайдите на https://replit.com
2. Создайте новый Repl (Python)
3. Загрузите проект через Git: `git clone https://github.com/IVANFROL/KikoGame.git .`
4. Установите зависимости: `pip install pygame`
5. Нажмите "Run" - игра запустится в браузере!
6. Получите публичную ссылку для сотрудников

Подробные инструкции: `REPLIT_SETUP.md`
