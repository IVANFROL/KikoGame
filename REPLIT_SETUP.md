# Инструкция по настройке KiKoGame на Replit

## Шаг 1: Создание Repl на Replit

1. Зайдите на https://replit.com
2. Войдите в аккаунт (или создайте новый)
3. Нажмите кнопку **"Create Repl"** (или "+ Create")
4. Выберите **"Python"** как язык программирования
5. Назовите проект: `KikoGame`
6. Выберите **"Public"** или **"Private"** (по вашему желанию)
7. Нажмите **"Create Repl"**

## Шаг 2: Загрузка файлов проекта

### Вариант A: Через Git (рекомендуется)

1. В Replit откройте консоль (Shell)
2. Выполните команды:
```bash
git clone https://github.com/IVANFROL/KikoGame.git .
```

### Вариант B: Через загрузку файлов

1. В Replit нажмите на иконку **"Files"** (слева)
2. Нажмите на три точки (⋮) рядом с именем проекта
3. Выберите **"Upload folder"**
4. Загрузите всю папку проекта с вашего компьютера

## Шаг 3: Установка зависимостей

1. В Replit откройте консоль (Shell)
2. Выполните команду:
```bash
pip install -r requirements.txt
```

Или напрямую:
```bash
pip install pygame
```

## Шаг 4: Запуск игры

1. Нажмите кнопку **"Run"** (зеленая кнопка вверху)
2. Игра запустится в окне справа
3. Если появится окно с игрой - всё готово!

## Шаг 5: Получение публичной ссылки

1. После запуска игры в правом верхнем углу будет кнопка **"Open in new tab"**
2. Или нажмите на иконку **"Webview"** (глобус)
3. Скопируйте ссылку - это и есть публичная ссылка для сотрудников!

## Возможные проблемы и решения

### Проблема: Игра не запускается
- Убедитесь, что установлен pygame: `pip install pygame`
- Проверьте, что файл `main.py` существует
- Проверьте консоль на наличие ошибок

### Проблема: Звуки не работают
- Это нормально для некоторых браузеров
- Игра будет работать, но без звука

### Проблема: Медленная загрузка
- Первый запуск может занять время
- Последующие запуски будут быстрее

## Дополнительные настройки

### Автоматический запуск при открытии Repl

Файл `.replit` уже настроен для автоматического запуска игры.

### Изменение размера окна

Если нужно изменить размер окна игры, отредактируйте файл `confi.py`:
```python
WIDTH = 1200  # измените на нужную ширину
HEIGHT = 700  # измените на нужную высоту
```
